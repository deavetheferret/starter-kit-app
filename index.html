Welcome on SC Kit 2.0

Note :

StarterKit app 2.0 is an evolution of the original starterkit CLI on npm.
If you want use the original instead of this one, no problem continue scroll ;).

Why StarterKit ?

Free and open source

Yes, starterkit 2.0 like his father is totally free and open source. You can submit your upgrades, bug fixes, enhancement and more.

A bunch of functionality

With StarterKit you can create a new directory with all the frontend stuff, like the index.html, the main.css, the main.js and Node.js.

Don't want to create a directory ?

No problem! You can also add the kit to an existing directory, all paths will be update and a gitignore will be create, cool huh?

Don't like the app...

No problem! I'll not cry but you can find the npm install just here.
